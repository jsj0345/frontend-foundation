폼 관련 태그

form 태그 내의 submit 버튼 클릭시 form에 작성된
입력값들을 서버로 전달하며 요청을 보내게 된다

form 속성 : action / method
action : 해당 form으로 요청을 보낼 경로
method : 요청 전송 방식을 지정하는 속성(get/post)
-get 방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식 (기본값)
-post 방식 : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식

form 태그 내의 제출(submit) 버튼 클릭시 action에 지정된 서버로 method에 작성한
요청 전송 방식으로 사용자가 입력한 값들을 key-value 세트로 전달한다.

라디오 버튼과 체크박스 관련 input 태그 타입

라디오 버튼(type="radio")

제시한 여러개의 값들 중 한개만 선택시켜야 할 때 사용
name 속성값이 같은 요소끼리 하나의 그룹으로 묶인다
제출시 현재 선택된(checked)값이 전달된다. 전달하고자 하는 값을 value 속성에 명시하기

성별

체크박스 (type="checkbox")

제시한 여러개의 값들 중 여러개를 선택 가능하게 할 경우
현재 선택된 checked값들이 전달된다
전달하고자 하는 값을 value 속성에 명시하기


select태그와 option태그

사용자에게 직접 키보드로 값을 입력받는게 아니라 내가 제시한 여러 옵션들 중에서 선택할 수 있게 해주는 태그
제출시 현재 선택된 selected 값이 전달된다
각각의 option 태그 value 속성값을 명시할 경우 value값이 전달되고
명시하지 않은 경우는 option 태그의 content 영역 값이 전달된다
content : 시작태그와 종료태그 사이에 작성된 텍스트
