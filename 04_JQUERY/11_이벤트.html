이벤트 핸들러(이벤트 발생시 실행될 function) 연결 방법

1)이벤트 메소드를 이용하는 방법

        $("선택자").이벤트메소드(function(){
           해당 요소에 해당 이벤트가 발생했을때 수행할 구문
        });

클릭



방법2. on 메소드 이용

        $("선택자").on("이벤트명",function(){
            실행시킬 구문
        });

마우스 클릭 또는 마우스를 올려보세요

방법3

        $("상위요소 선택자").on("이벤트명","하위요소 선택자",function(){
            선택한 상위요소 안에 존재하는 하위요소 이벤트가 발생했을때 수행 하는 내용


        });

키보드 관련 이벤트

keydown/keypress : 키가 눌려질때 발생하는 이벤트
keydown : 키보드의 모든 키가 눌려질때 발생
keypress : 키보드의 펑션키, 기능키등등 한글 제외 나머지 키가 눌렸을 때
keyup : 키가 눌렸다 떼어질때 발생
