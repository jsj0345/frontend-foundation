Content 영역(innerHTML,innerText) 관련 메소드

html('문구') 메소드 : innerHTML 속성과 관련된 메소드

선택된 요소의 content영역(시작태그와 종료태그 사이)을 리턴해주거나 또는 변경해주는 메소드
-$('선택자').html() : 선택된 요소의 content영역의 값을 반환해준다. (이때 html태그까지 포함)
-$('선택자').html('문구') : 선택된 요소의 content 영역값을 해당 문구로 변경해준다
(전달하는 문구에 html 태그가 있을 경우 태그로써 해석한다.)

*text('문구') 메소드 : innerText 속성과 관련된 메소드

선택된 요소의 content영역의 텍스트만 리턴해주거나 변경해주는 메소드
$("선택자").text() : 선택된 요소의 content 영역을 반환해줌(단, html 태그는 무시하고 텍스트만)
$('선택자').text('문구') : 선택된 요소의 content 내용을 전달값으로 변경해줌 (단, html 태그는 해석불가)
