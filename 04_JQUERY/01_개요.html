JQUERY란?

기존에 자바스크립트 언어만으로 구현하기에
복잡했던 구문 작성을 간소화 하기 위해 개발된 javascript 기반 라이브러리
javascript 언어로 유용한 함수나 내용들이 정의되어있는 도구 모음

복잡한 자바스크립트 구문을 보다 빠르고 쉽게 개발 가능
적은 양의 코드로 풍부한 기능 제공
DOM : HTML 문서를 객체로 표현한 구조. jquery 장점
1)DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있다.
2)Ajax (비동기통신), 이벤트 처리 등 폭넓게 지원
3)jQuery와 같은 확장형 플러그인 오픈 API를 지원한다.

*jQuery 구문 작성

jquery 구문으로 DOM요소를 다루는 구문을 기본적으로 작성
jquery 명령은 문서상의 요소들이 다 만들어진 후에 실행되어야한다.

-자바스크립트에서 해당 구문은 

            window.onload = function() {
                실행내용;
            }

            onload : 해당 요소가 읽혀지면 동작하는 이벤트
            단, javascript에서의 window.onload 구문은 해당 문서의 요소들도 다 로드되고
            연동된 외부 문서들도 로드된 이후에 실행된다. 
            또한 window.onload는 해당 문서에서 한번만 작성 가능하다.
jQuery에서의 위와 같은 구문은 3가지 형태로 작성가능하다. 
            1) jQuery(document).ready(function(){
                실행내용;
            });

            2) $(document).ready(function(){
                실행내용; 

            });

            3)$(function(){
                실행내용;
            });

            $는 jquery를 나타내는 문자로
            위 구문은 DOM요소가 다 로드되면 실행되는 구문으로 여러번 기술도 가능하다.
