자바스크립트 개요

1.스크립트 언어란?

프로그램 동작을 사용자(클라이언트)의 요구에 맞게 수행되도록 해주는 용도
빠르게 배우고 짧은 소스코드로 상호작용 되도록 고안되었다.


2.자바스크립트란?

웹 브라우저에서 가장 많이, 가장 기본적으로 사용되는 인터프리터 방식의 스크립트 언어
-자바: 컴파일방식(파일 전체를 컴파일 거치고 나서 실행됨) -자바스크립트 : 인터프리터 방식 - 파일 전체가 아닌 코드 한줄씩 읽고 처리된다

3. 장단점

장점

  • 요소에 이벤트가 발생했을 때 코드를 한줄씩 읽어 나가며 수행되기 때문에 수행 속도가 빠르다.
  • 간단한 코드 작성으로 초보자가 접하기 쉬운 언어 (단순한 구조, 원칙)
    ex)접근제한자 개념 없음, 메소드 정의시 반환형 정의하지 않음, 변수 선언시 자료형 없음 등등..

단점

  • 웹에 특화된 기술 - 내부에서 제공하는 기능이 제한적이다.
  • HTML 내에 소스코드를 작성하게 되면 외부에 공개된다.

4.자바스크립트 소스코드 작성 위치에 따른 종류

  1. inline(인라인) : 이벤트를 부여하고자 하는 요소 내에 간단한 소스코드를 작성하여 실행되게 하는 방법
  2. internal(내부) : 해당 html 문서 내에 script 태그 영역에 소스코드를 작성하여 실행되게 하는 방법
  3. external(외부) : 소스코드들을 별도의 .js 파일로 작성하여 불러와 실행하는 방법

1) inline 인라인 방식

태그 내에 직접적으로 실행할 간단한 소스코드를 작성하여 실행시킨다
주로 실행해야하는 코드가 소량일 경우 사용된다
[표현법] <태그명 이벤트속성="해당 이벤트 발생시 실행 코드">

2)internal 내부 방식

script 태그 영역에 함수 단위로 소스코드들을 작성해놓고
특정요소에 이벤트가 발생하면 호출될 수 있도록 작성하고 실행하는 방식
script 태그는 head/body 태그 내에 작성 가능

3)external 외부 방식

별도의 .js파일로 소스코드를 작성하고 해당 html에 불러와 실행하는 방법
