2.데이터를 입력받는 기본 구문(변수에 기록 가능)

1) 변수 = window.confirm('질문내용');
2) 변수 = window.prompt('질문내용');
3) 변수 = 선택한요소.속성;
4) 변수 = 선택한input요소.value; (제일 중요)

1)window.confirm('질문내용')

confirm 호출시 "질문내용" 과 확인/취소 버튼이 존재하는 알림창 발생
확인 버튼 클릭시 true/ 취소 버튼 클릭시 false를 반환한다.

2)window.prompt("질문내용")

prompt 호출시 질문내용과 입력할 수 있는 입력상자와 확인/취소 버튼이 보여지는 알림창 발생
-확인 버튼 클릭시 텍스트 상자에 입력되어 값이 변환
-취소 버튼 클릭시 null을 반환
