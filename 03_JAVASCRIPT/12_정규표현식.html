정규 표현식

정규 표현식 (REGEX / REGEXP : Regular Expression) 이란
특정 패턴을 가진 문자열을 찾거나 변경(치환)할때 사용하는 형식 언어
정규표현식을 이용하면 문자열에 대해 특정 패턴 조건 검사시 또는 치환시
복잡한 조건을 제시할 필요 없이 간단하게 처리 가능하다
자바스크립트 뿐만 아니라 자바나 오라클 등 다른 언어에서도 사용 가능한 방식

정규 표현식 객체 생성 및 정규 표현식과 관련된 메소드

*메타문자

메타문자를 이용해서 정규표현식으로 검색 조건을 삼을 특정 패턴 제시 가능 문자열이 해당 정규식으로 제시된 패턴을 만족하는지
test()를 통해 검사하거나 replace를 이용하여 치환가능

*플래그문자

/패턴/g 와 같이 패턴 / 뒤에 작성
i : 대소문자를 가리지 않고 비교 수행
g : 문자열 내의 모든 패턴을 찾기 (전역적으로 비교)
m : 여러줄 비교 수행

*추가적인 메타문자

\d : 숫자 ([0-9]랑 동일)
\D : 숫자를 제외한 모든 문자
\w : 영문자,숫자,_
\W : 영문자,숫자,_를 제외한 모든 문자
\s : 공백문자 (띄어쓰기,탭,줄바꿈)
\S : 공백문자를 제오한 모든 문자

*수량문자들
+ : 1글자 이상 (최소 한글자)
* : 0글자 이상 (0글자 가능)
? : 0글자 또는 1글자 (2글자 이상 불가능)

{5} : 5글자
{2,5} : 2글자이상 5글자 이하
{2,} : 2글자 이상
{,5} : 5글자 이하
