padding

내용물 영역(content)과 테두리 영역(border) 사으이 거리(여백)을 조절하는 속성



컨텐츠 영역
padding:100px
컨텐츠 영역
padding-top:50px; padding-left:100px;
컨텐츠 영역
padding: 50px 20px 10px 0px;

margin

타 요소들 간의 간격을 조절할 때 사용

컨텐츠 영역
margin:50px;

컨텐츠 영역
margin-top:50px; margin-left:20px;

컨텐츠 영역
margin: 50px 30px 20px 0px;
