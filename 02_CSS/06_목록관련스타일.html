목록 관련 스타일

list-style-type : 불릿 기호를 변경시켜줄때 사용

        *순서 없는 목록(ul)
        선택자 {
            list-style-type:disc/circle/square/none;
        }

        *순서 있는 목록(ol) 
        선택자 {
            list-style-type: decimal / decimal-leading-zero;
            list-style-type: lower-alpha/upper-alpha;
            list-style-type: lower-roman/upper-roman
        }

순서 없는 목록

  • HTML
  • CSS
  • JAVASCRIPT
  • JQUERY

순서 있는 목록

  1. HTML
  2. CSS
  3. JAVASCRIPT
  4. JQUERY

list-style-position : 불릿 기호 위치를 조절할 때 사용

list-style-image : 불릿 기호로 이미지를 적용할 때 사용

        선택자 {
            list-style-position : inside/outside;
            list-style-image : url (적용할 이미지 경로); 
        }
