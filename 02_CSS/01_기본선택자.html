CSS 기본 선택자

선택자란 ?

특정 HTML 요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택하여 원하는 스타일과 기능을 적용시킬 수 있다.

1.모든(전체) 선택자 *

모든 요소를 다 선택하고자 할때 사용 

    * {
        스타일속성:속성값;
        ...
    }

2.태그 선택자 (태그명)

해당 문서 내에 같은 태그를 모두 선택할 때 사용 

    태그명{
        스타일속성 : 속성값;
        스타일속성 : 속성값;
    }





3.클래스 선택자(.클래스명)

해당 문서 내에 있는 원하는 요소 여러개를 선택하고자 할 때 사용
해당 요소에 클래스를 부여해야한다.
class="클래스명" 

    .클래스명{
        스타일속성 : 속성값;
        ...
    }

4.아이디 선택자(#아이디명)

해당 문서 내에 특정 HTML 요소 하나만을 선택하고자 할 때 사용
단 해당 요소의 id값은 문서내에서 고유한 값을 가져야한다(식별자) 다른 곳에 동일 아이디를 주면 스타일에 따라 변경은 되겠지만 중복되지 않게 사용하는게 관례이다. (중복되게 사용하고싶다? 클래스를 사용하기) 

    #아이디명 {
        스타일속성: 속성값;
    }
