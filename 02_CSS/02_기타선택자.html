1.속성 선택자

스타일을 부여하고자 하는 요소 내에 작성되어있는 속성을 이용하여 선택하는 방법
스타일 뒤에 []를 이용하여 속성과 속성값을 제시하여 선택한다

[작성법]
선택자[속성=속성값] : 해당 속성값을 가질때
선택자[속성~=속성값] : 특정 단어를 속성값으로 가질 때 (단어별 - 와 띄어쓰기 기준)
선택자[속성|=속성값] : 특정 단어를 속성값으로 가질 때 (시작단어 유무)
선택자[속성^=속성값] : 속성값이 특정값으로 시작할 때
선택자[속성$=속성값] : 속성값이 특정값으로 끝날 때
선택자[속성*=속성값] : 속성값에 특정 요소가 포함될 때

2.자손 선택자와 후손선택자

자손 : 바로 하위 요소
후손 : 하위 요소들 전부 

        자손 선택자 [ > ]
        a>b{
            스타일속성 : 속성값;
        }


        후손 선택자 [  ]  - 공백표현
        a  b{
            스타일속성 : 속성값;
        }

3.동위(같은 위치) 선택자

동위관계(같은 레벨)에 있는 요소를 사용할 때 사용 

        ** a요소 뒤에 있는 b요소 하나만 선택

        a+b{
            스타일 속성 : 속성값;
        }

        **a요소 뒤에 있는 모든 b요소 선택

        a~b{
            스타일속성 : 속성값;
        }

4.반응 선택자

사용자의 움직임에 따라 선택이 되거나 안되는 선택자 

        해당 요소가 클릭되었을 때 스타일을 부여하는 선택자
        선택자:active{
            스타일 속성 : 속성값;

        }

        해당 요소에 마우스가 올라갔을때 내려갔을때 스타일 부여 선택자
        선택자:hover{
            스타일 속성 : 속성값;
        }

5.상태 선택자

요소의 상태에 따라서 선택되는 선택자 

        1)checked(체크된) 상태의 요소

        선택자: checked{
            속성명 : 속성값;
        }
    

        2)초점(focus)이 맞춰진 input 요소 선택자

        선택자:focus{
            스타일속성:값;
        }
