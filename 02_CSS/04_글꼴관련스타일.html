font-size : 글꼴의 크기를 변경할 때 사용

        선택자 {
            font-size : 크기(px/em/%)
            
        }
  • 고정 크기 px 테스트
  • 가변 크기 em 테스트
  • 가변 크기 % 테스트

font-weight : 글꼴 굵기 변경

        선택자 {
            font-weight : normal/bold/bolder/lighter/100/400/900;
        }
  • 원래 굵기
  • 굵은 글꼴
  • 원래 굵기보다 굵게
  • 원래 굵기보다 가늘게

        }

  • 글꼴을 기울임
