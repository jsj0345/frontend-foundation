태그가 변형 속성에 의해 변형될때 부드럽게 변형될 수 있도록 적용하는 속성
transition-property : 적용할 속성;
transition-duration : 지속시간;
transition-timing-function : linear,ease,ease-in,ease-out; 동작 느낌
transition: 한번에 모든 속성 적용;

linear : 균일

ease : 극초반 느리게 초반 빠르게 끝 느리게

ease-in : 점점 빠르게

ease-out : 점점 느리게

ease-in-out : 시작지점 종료지점 천천히
